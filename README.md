# Cocktail lover(ALCOHOLIC)들을 위한 놀이터
#### SNS 기반 칵테일 소개, 추천 서비스

## 📢 프로젝트 소개
칵테일을 좋아하는 Z세대를 위한 소셜 네트워킹 사이트

### 배경
현대 사회에서 칵테일 문화는 빠르게 확산되고 있습니다. 특히 Z세대는 새로운 맛과 경험을 추구하며, SNS를 통해 자신의 경험을 공유하는 것을 즐깁니다. 이러한 트렌드를 반영하여, 칵테일 애호가들이 모여 자신의 레시피를 공유하고, 다른 사람들의 추천을 받아볼 수 있는 플랫폼을 개발하게 되었습니다.

### 중요성
기존의 칵테일 관련 플랫폼들은 주로 정보 제공에 집중되어 있어, 사용자 간의 상호작용이 제한적입니다. 이 프로젝트는 사용자들이 서로의 레시피와 경험을 공유하며, 더 나은 칵테일 문화를 형성하는 데 중점을 둡니다. 이를 통해 칵테일 입문자들도 쉽게 접근할 수 있으며,독창적인 레시피를 접할 수 있는 기회를 제공합니다.

### 주요 사용자 그룹
- **칵테일 입문자**: 기본적인 칵테일 레시피와 재료에 대한 정보를 쉽게 얻고, 추천받을 수 있습니다.
- **칵테일 애호가**: 자신만의 독특한 레시피를 공유하고, 다른 애호가들과 소통할 수 있습니다.

## ⛳ 프로젝트 목표
- **칵테일 입문자들을 위한 접근성 향상**
  - 입문자들이 쉽게 따라 할 수 있는 기본적인 레시피와 재료 정보를 제공하여, 칵테일 문화를 쉽게 접할 수 있도록 합니다.
  
- **사용자가 서로의 레시피와 경험을 공유함으로써 더 나은 칵테일 문화를 형성**
  - 사용자들이 자신만의 레시피와 경험을 공유하고, 서로의 피드백을 통해 더 나은 레시피를 개발할 수 있도록 지원합니다.
  
- **커뮤니티 형성을 통한 지속 가능한 사용자 참여 유도**
  - 칵테일에 대한 다양한 주제의 토론과 정보 교류가 이루어질 수 있는 커뮤니티를 형성하여, 사용자들이 지속적으로 참여하도록 유도합니다.

- **독창적인 레시피와 창의성 발휘**
  - 사용자들이 창의적인 아이디어를 통해 독특한 칵테일 레시피를 개발하고 공유할 수 있는 공간을 제공합니다.

- **긍부정 모델을 통한 건전한 소통 환경 조성**
  - 댓글의 긍부정 분석을 통해 건전하고 긍정적인 소통 환경을 조성하며, 사용자들이 보다 객관적인 피드백을 받을 수 있도록 합니다.

## 😀 프로젝트 맴버 역할 분담

| 이름   | 담당 업무                   |
| ------ | --------------------------- |
| 👑오준하 | 팀장/Back-End 개발/긍부정 모델 학습 및 연동 |
| 📅김수민(PM) | Front-End 디자인 및 화면구현 |
| 전다혜 | Front-End 디자인 및 화면구현 |
| 김민석 | Back-End 개발/데이터 전처리 |
| 노한울 | Back-End 개발/데이터 분석,전처리 |

### 상세 역할 분담
- **팀장: 오준하(Back-End)**
  - 텍스트 긍부정 모델 생성
  - DB 구축
  - 재료 페이지,메뉴 페이지 기능 구현
  - 마이 페이지,게시판 기능 구현
  - 피드, 찜 기능 구현

- **팀원: 김수민(Front-End, PM)**
  - 홈페이지 로고 제작
  - 로그인 & 회원가입,
  - HOME, INGREDIENT, BOARD, MYPAGE 화면 구현
  - 검색, 간단 추천 페이지 화면 구현
  - 시연 영상 제작 및 문서 작업

- **팀원: 전다혜(Front-End)**
  - MENU, FEED, 댓글창 화면 구현
  - 소개서 작성 및 전반적인 화면 디자인
  -  PPT 제작
 
- **팀원: 김민석(Back-End)**
  - 회원가입, 로그인
  - 데이터 전처리 , DB 구축
  - 이미지 수정,검색 및 간단 추천 페이지 기능 구현

- **팀원: 노한울(Back-End)**
  - 데이터 수집 및 전처리 , DB 구축
  - 게시판 화면 구현
  - 간단 추천 페이지 데이터 분석 및 구현
